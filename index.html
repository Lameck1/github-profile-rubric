Assess your GitHub profile - Rubric

Instructions

  1. Make a copy of this document by selecting “File” and then “Make a copy”.
  2. For each requirement, analyze your GitHub profile to see if it meets the requirement.
  1. Identify and highlight the score that best describes that section of your GitHub profile.
  2. Write down any areas of improvement in a comment in this same column.
  3. Refer to the learning resources for further study.
  1. Make the required changes by implementing all of your own comments to improve your GitHub profile.
  2. Continue with the next set of requirements.
